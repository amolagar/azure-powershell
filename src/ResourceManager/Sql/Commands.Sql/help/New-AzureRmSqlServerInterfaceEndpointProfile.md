# New-AzureRmSqlServerInterfaceEndpointProfile

## SYNOPSIS
Creates an Azure SQL Server Interface endpoint profile

## SYNTAX

```
New-AzureRmSqlServerInterfaceEndpointProfile -InterfaceEndpointProfileName <String> -VirtualNetworkSubnetId <String>
 [-AsJob] -ServerName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Creates an Azure SQL Server Interface endpoint profile. 

## EXAMPLES

### Example 1
```
PS C:\> $interfaceEndpointProfile = New-AzureRmSqlServerInterfaceEndpointProfile -ResourceGroupName rg -ServerName serverName -InterfaceEndpointProfileName interfaceEndpointProfileName -VirtualNetworkSubnetId virtualNetworkSubnetId
```

Creates an Azure SQL Server Interface endpoint profile
